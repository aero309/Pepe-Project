Pepe Mining Simulator

Description

Pepe Mining Simulator est un jeu incrémental (Anglais: Clicker Game). La mécanique de ce jeu consite de simples actions telles que les "clicks" à répétiton sur un bouton. Le joueur peut également laisser le jeu s'exécuter sans rien faire.

Instructions

Avec votre souris, cliquez sur le personnage pour gagner des Pepe Coins. Avec le solde que vous aurez accumulé, vous pourrez ainsi profiter des différentes améliorations telles que:

  • Upgrade coins per click: Ajout d'incréments
  • Upgrade cooldown: Reduire le temps entre chaque "click"
  • Pepe Assistants: Ajout des "auto-clicks"
